
===========================
How to Use 1-my_list.py
===========================

This module only contains the class My_List "class MyList(list):".

=====
Usage
=====

The class inherits from list, usage is exactly the same as any other list

=====
Tests
=====

    >>> MyList = __import__('1-my_list').MyList

1   Checks instantation and append

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> print(my_list)
    [1, 4, 2]

2   Checks my_list inherits from list

    >>> my_list = MyList()
    >>> print(issubclass(MyList,list))
    True


3   Ordered list sorting

    >>> test_list = MyList()
    >>> test_list.append(1)
    >>> test_list.append(2)
    >>> test_list.append(3)
    >>> test_list.append(4)
    >>> print(test_list)
    [1, 2, 3, 4]
    >>> test_list.print_sorted()
    [1, 2, 3, 4]

4 Unordered list sorting

    >>> test_list = MyList()
    >>> test_list.append(4)
    >>> test_list.append(8)
    >>> test_list.append(1)
    >>> test_list.append(0)
    >>> print(test_list)
    [4, 8, 1, 0]
    >>> test_list.print_sorted()
    [0, 1, 4, 8]

5 Unordered list with negative numbers sorting

    >>> test_list = MyList()
    >>> test_list.append(4)
    >>> test_list.append(8)
    >>> test_list.append(-1)
    >>> test_list.append(-4)
    >>> print(test_list)
    [4, 8, -1, -4]
    >>> test_list.print_sorted()
    [-4, -1, 4, 8]
